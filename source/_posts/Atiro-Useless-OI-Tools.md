测试本地样例，可以：

```bash
$ atiro judge A
```

你甚至可以用上 checker、interactor、solver 和 generator：

```bash
$ atiro j --judg checker --grad grader --solv solver --make maker.cpp
```

OJ Tools，拉取样例、提交代码：

```bash
$ atiro cf get https://codeforces.com/contest/1852/problem/B
```

```bash
$ atiro cf submit TEST
```
